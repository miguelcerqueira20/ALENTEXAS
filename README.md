# ALENTEXAS Countdown

Mini-site estático pronto para GitHub Pages.

## O que inclui

- Ecrã inicial com botão **Entrar e começar música**
- Countdown até **23 de julho de 2026**
- Player Spotify com a playlist `2OfgaAqnqTT55NSZGFM4HT`
- Slideshow automático com fotos e vídeos
- Vídeos sem som, já convertidos para MP4 sem áudio
- Frase no slideshow: **um cheirinho do que será este ano**
- Secção manifesto ALENTEXAS

## Como publicar no GitHub Pages

1. Cria um repositório no GitHub.
2. Faz upload de **todos os ficheiros e pastas que estão dentro deste ZIP**:
   - `index.html`
   - `style.css`
   - `script.js`
   - pasta `media`
   - `README.md`
3. Vai a **Settings > Pages**.
4. Em **Build and deployment**, escolhe **Deploy from a branch**.
5. Seleciona `main` e `/root`.
6. Guarda.

## Nota sobre a música Spotify

O botão abre sempre o countdown. A tentativa de iniciar música pelo Spotify depende das regras do browser e do próprio Spotify. Se o Spotify bloquear o arranque automático, o player fica visível e basta carregar no play.
